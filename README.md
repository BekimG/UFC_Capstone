# UFC Fight Predictor
This repositiory contains my capstone project done at Digital Futures - UFC Fight Predictor
This is a predictive modelling project, in which based on fight and fighter statistics is able to output the winner of a fight, between 2 UFC fighters.
Outcomes:
* Created an XGBoost model, which predicts the winner, with an accuracy of 69.8%
* Created a Streamlit App for users to interact with the model

Data Source: https://www.kaggle.com/datasets/neelagiriaditya/ufc-datasets-1994-2025
